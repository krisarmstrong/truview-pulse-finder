# Contributing to BigRed WebSocket Client

## Prerequisites

- Python 3.6+
- websockets library
- NetScout nGeniusPULSE devices (for testing)

## Setup

1. Clone repository
2. Install: `pip install -r requirements.txt`
3. Run tests

## Development

- Follow PEP 8
- Test WebSocket communication
- Verify both English and Spanish output
- Test MAC filtering
- Check detail levels
